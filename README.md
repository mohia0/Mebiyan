# SVG Motion Tool

A powerful web-based tool for creating animated SVG graphics with various motion effects.

## Features

- Multiple motion types: Swap, Snap, Fade, Robot, and Pair
- Adjustable speed and size controls
- Color customization (White/Black)
- GIF export functionality
- Interactive SVG preview selector
- Drag and drop SVG positioning

## Setup for GitHub Pages

To enable GitHub Pages for this repository:

1. Go to your repository on GitHub: https://github.com/mohia0/Mebiyan
2. Click on **Settings** (top menu)
3. Scroll down to **Pages** (left sidebar)
4. Under **Source**, select:
   - **Branch**: `main`
   - **Folder**: `/ (root)`
5. Click **Save**
6. Wait a few minutes for GitHub to build and deploy your site
7. Your site will be available at: `https://mohia0.github.io/Mebiyan/`

## Files

- `index.html` - Redirects to the main tool
- `svg_motion_tool.html` - Main application
- `Opt 2/` - SVG files used by the tool

## Usage

Simply open `svg_motion_tool.html` in a web browser or visit the GitHub Pages URL once configured.
